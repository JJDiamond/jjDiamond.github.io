三栏自适应布局效果

当今社会，随着移动设备的增多，对于页面的自适应要求越来越高，其中，我们经常遇到的效果就属于的三栏布局，在这里，给出我熟知的三种布局方式，分别为：绝对定位方式，margin负值处理和浮动定位

绝对定位方法

html

css

这里的左中右三个div的顺序是可以任意调整的，这与剩下的两中方法就不一样了，需要注意一下。
此方法的优点是，理解容易，上手简单，受内部元素影响而破坏布局的概率低，就是比较经得起折腾。
缺点在于：如果中间栏含有最小宽度限制，或是含有宽度的内部元素，当浏览器宽度小到一定程度，会发生层重叠的情况。然而，一般情况下，除非用户显示器分辨率宽度>=1600像素，否则用户不会把浏览器缩小到1000像素以下的。
